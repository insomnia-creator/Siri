# Building yourself

Requirements:
1. A computer.
2. A brain.
3. Node.js(lts works fine)
4. A terminal

Clone this repository into your own.

```zsh
git clone https://github.com/insomnia-creator/Siri.git
```
Run `npm i` to install the modules.


And run ``npm start`` to start the bot.
